进出校审核管理系统 - 使用说明
================================

一、系统概述
------------
这是一个用C语言编写的进出校审核管理系统，包含学生和辅导员两种角色。

二、文件结构 (Version 4 - 模块化版本)
------------
1. main.c                 - 主程序入口
2. common.h               - 公共头文件（数据结构、函数声明、常量）
3. data_management.c      - 数据文件管理函数
4. user_auth.c            - 用户认证和登录函数
5. student_operations.c   - 学生操作函数
6. advisor_operations.c   - 辅导员操作函数
7. application_handling.c - 申请处理函数
8. school_management.exe  - 编译后的可执行文件
9. users.dat              - 用户数据文件
10. applications.dat      - 申请记录数据文件

三、编译方法
------------
模块化编译命令：
gcc main.c data_management.c user_auth.c student_operations.c advisor_operations.c application_handling.c -o school_management.exe

或者使用批处理脚本编译：
compile.bat

三、运行方法
------------
方法1（推荐）：双击运行 test_system.bat
方法2：在命令行中运行 school_management.exe

五、系统功能
------------
1. 登录系统：
   - 输入姓名、学工号、密码和身份进行登录

2. 学生菜单：
   - 提交出校申请（需填写出校时间、回校时间、原因、目的地、联系方式）
   - 查看申请记录（含审批状态）
   - 切换账号
   - 退出系统

3. 辅导员菜单：
   - 查看待审批名单（可查看学生详细信息并审批）
   - 查看审批记录
   - 切换账号
   - 退出系统

六、注意事项
------------
1. 程序已内置中文乱码修复功能，可直接运行 school_management.exe
2. 如果仍有中文显示问题，请使用 test_system.bat 运行程序
3. 系统数据存储在 users.dat 和 applications.dat 文件中
4. 首次运行时会自动生成测试数据
5. 时间格式建议：YYYY-MM-DD HH:MM


八、示例使用流程
----------------
1. 学生登录 -> 提交申请 -> 等待审批
2. 辅导员登录 -> 查看待审批名单 -> 审批申请
3. 学生再次登录 -> 查看申请状态
